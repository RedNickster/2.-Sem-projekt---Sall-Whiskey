# Sall Whisky Distillery - Lagerstyringssystem

**Projekt:** IT-system til digital lagerstyring og sporbarhed for Sall Whisky, et lokalt destilleri i Sall, Hammel.  
**Udviklere:** Luca, Nicklas, Lucas (25T, 2026).  
**Formål:** Erstatte manuelle logbøger med et system til registrering af fade, destilleringer, og whiskyprodukter, inkl. automatisk beregning af svind (Angels’ Share/Devils’ Cut) og QR-kode-sporbarhed.

**Arkitektur:** 3-lags model (Presentation, Logic, Storage) i Java.  
**Kernefunktioner:** Registrering af fade, destilleringer, lageroversigt, og whiskyhistorik.  
**Teknologier:** Java (backend), lav kobling og høj samhørighed.

**Projektperioder:**

- Inception (23/04–29/04): Forretningsanalyse.
- Elaboration + Construction (29/04–20/05): Arkitektur, use cases, kodning, test, dokumentation.
  - Lavet iderativt og så derfor er Elaboration og Construction sket "sammentidig"
